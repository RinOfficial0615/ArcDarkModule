# ArcDarkModule

Replace track images of the certain rhythm game to dark ones

## Requirements

- Android NDK >= r28
- Phone with zygisk enabled

## How to use?

- Clone the repository
- Set `ANDROID_NDK_HOME` environment variable
- Open your powershell and run `.\build.ps1 --rel`
- Push `.\outputs\ArcDarkModule.zip` to your phone and install it in your root manager
- Enjoy!
